# 🌐 Personal Portfolio Website

🚀 **Live Demo:** https://portfolio-main-xi-self.vercel.app/

---

## 📌 About the Project

This is a **personal portfolio website** designed to showcase my skills, projects, and achievements in a clean and professional way.
The website highlights my journey as a developer and provides an easy way to connect with me.

The portfolio is fully responsive and optimized for all devices, ensuring a smooth user experience.

---

## ✨ Features

* 👩‍💻 Modern and responsive UI design
* 📂 Projects showcase section
* 📄 Resume / profile display
* 📬 Contact form integration
* 🎯 Smooth navigation and animations
* ⚡ Fast performance using modern web technologies

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Framework:** Next.js
* **Styling:** Tailwind CSS
* **Deployment:** Vercel

### Clone the repository

```bash
git clone https://github.com/anishasuman/portfolio-main.git
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```
